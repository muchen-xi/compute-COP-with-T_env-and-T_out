# 空气源热泵 COP 建模

本项目基于某型号空气源热泵的实测运行数据，建立了 COP（性能系数）与环境温度、出水温度之间的数学模型。通过枚举法求解六种候选模型的最优系数，并比较其拟合优度（R²），最终确定最佳模型形式。

## 项目简介

数据来源于 Excel 表格，包含不同环境温度（-26℃ 到 21℃）和出水温度（35℃ 到 55℃）下的制热量与输入功率。  
程序完成以下任务：

1. 读取并预处理数据，计算 COP。
2. 针对六种模型（二次/一次 × 线性/平方根/对数）分别进行：
   - 遍历所有可能的 4 点组合（二次模型）或 3 点组合（一次模型），求解系数。
   - 用每组系数预测全体数据，计算 R²，保留最优系数。
3. 比较六种模型的最优 R²，选出整体最佳模型，并输出带系数的公式。

## 运行环境

- Python 3.7+
- pandas
- numpy
- openpyxl
- tqdm（可选，用于显示进度条）

安装依赖：

```bash
pip install pandas numpy openpyxl tqdm

```
## 使用方法
1. 准备数据:
- 将 Excel 数据文件放在 data/ 目录下（或修改代码中的路径），并确保文件名为 data.xlsx。
- 数据格式必须与示例文件一致：
- 第 2 行：环境温度数值（-26, -20, -15, ...）
- 第 3 行：标签（“热量kW”、“功率kW”交替）
- 第 4~8 行：不同出水温度（35℃, 40℃, 45℃, 50℃, 55℃）对应的热量和功率数据

2. 运行程序
